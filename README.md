# 🔐 IntelliSecure

A secure role-based backend application built using FastAPI, JWT authentication, and MySQL.

---

## 📌 Project Overview

IntelliSecure is a backend authentication and authorization system designed to manage multiple user roles securely using JWT tokens.

---

## 🚀 Features

- User Authentication
- JWT Login
- Role Based Access Control (RBAC)
- FastAPI REST APIs
- MySQL Database
- Middleware Authentication
- Secure Password Handling
